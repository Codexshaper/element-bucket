== Description ==

CodexShaper Element Bucket is an all-in-one addon for Elementor that supercharges your website design with over 90 custom widgets.

== Frequently Asked Questions ==

= How do I install CodexShaper Element Bucket? =

Go to your WordPress dashboard -> Go to Plugins -> Click on 'Add new'-> In the Search field, enter CodexShaper Element Bucket and choose CodexShaper Element Bucket.

Press install -> After installation, click Activate.

Note: Make sure your are installed Elementor plugin to use CodexShaper Element Bucket.
